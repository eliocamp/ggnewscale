# ggnewscale: Multiple Fill and Colour Scales in 'ggplot2'

Use multiple fill and colour scales in 'ggplot2'.

## Usage

To have more than one colour or fill scale in a plot using ggplot2, you
add a new scale slot with
[new_scale](https://eliocamp.github.io/ggnewscale/reference/new_scale.md)
and the following geoms will use a different scale than the previous
ones.
